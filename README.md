## Women&apos;s Day – Interactive Tribute Website

This project is a small, interactive single–page website created to celebrate International Women&apos;s Day. It highlights women&apos;s stories, milestones across generations, and invites visitors to write a personal promise for a woman who inspires them.

Built with **React** and **Vite**, the page uses a soft pink–and–white theme, smooth hover animations, and a reflective "Who are you celebrating today?" section.

### Features
- Hero section with a spotlight area that changes text based on the role you select (Leaders, Creators, Changemakers, Everyday).
- Story, journey, and impact sections that explain different ways women shape the world.
- An interactive Celebrate area where you can type a name and a message, then see your promise reflected back to you.

### Tech Stack
- React
- Vite
- CSS (custom styling, no UI framework)

### Getting Started
1. Install dependencies:
	- In the inner project folder `Women's Day/Women's Day` run: `npm install`
2. Start the development server:
	- From the same folder: `npm run dev`
3. Open the URL shown in the terminal (for example `http://localhost:5173/` or `http://localhost:5174/`).
